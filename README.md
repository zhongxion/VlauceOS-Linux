# VluaceOS V1

## 项目简介

**VluaceOS** 是一款从底层开始完全自主构建的企业级 Linux 发行版。它并非任何现有系统的"换皮"或"二次打包"，而是一套拥有完整独立基础设施的原创操作系统。


## 🧬 完全自研的技术栈

### 1. 独立软件仓库系统

VluaceOS 拥有**完全自建的软件仓库**，所有软件包均由官方团队构建、签名并分发：

| 组件 | 说明 |
|:---|:---|
| 仓库服务器 | 独立部署，支持 `current` / `nonfree` / `multilib` / `debug` 四类源 |
| 签名体系 | GPG 密钥签名，确保包来源可信、传输完整 |
| 更新策略 | 安全更新定期移植审计，不依赖第三方源 |

用户无需依赖任何外部软件源，所有软件均可从 VluaceOS 官方仓库获取。

### 2. 自研核心组件

- **`vgt`（VluaceOS Grand Tool）**：自研的主程序封装工具，作为系统的统一命令行入口，管理包、服务、系统配置等核心操作。
- **`runci` 中间层劫持**：自主研发的命令拦截与审计框架，提供细粒度的系统调用监控和访问控制能力。

### 3. 完整的品牌独立性

| 层面 | 实现 |
|:---|:---|
| 系统标识 | 独立 `/etc/os-release`，ID 为 `skmoyt` |
| 版本体系 | 版本号 `V1`，代号 `skmoyt` |
| 视觉识别 | 全平台 ASCII 字符画、终端提示符、登录横幅均为原创设计 |
| 引导界面 | GRUB 菜单、启动画面完全去除第三方商标 |


## 🏗️ 企业级架构设计

VluaceOS 采用 **Void Linux musl** 作为底层基础，在此之上构建了完整的企业级能力层：
┌─────────────────────────────────────────────┐
│ VluaceOS 应用层 (vgt/runci/自研工具) │
├─────────────────────────────────────────────┤
│ VluaceOS 软件仓库 (自建 + GPG 签名) │
├─────────────────────────────────────────────┤
│ Void Linux musl (经过深度定制与去品牌化) │
├─────────────────────────────────────────────┤
│ Linux 内核 │
└─────────────────────────────────────────────┘

### 核心特性

| 特性 | 说明 |
|:---|:---|
| **轻量高效** | 基于 musl C 库，系统占用极低，启动迅速 |
| **企业级工具链** | 内置 GCC、Python、Java 17、Go、Node.js 等完整开发环境 |
| **LAMP 支持** | Apache + MariaDB + PHP 8.5 全套 Web 服务栈 |
| **安全加固** | iptables + nftables 双防火墙，支持 fail2ban |
| **服务管理** | runit 服务管理，轻量可靠 |
| **容器就绪** | 预装 Docker / Podman / Buildah 容器工具链 |

## 📌 项目定位

VluaceOS 面向**企业级服务器**和**云计算场景**，强调：
